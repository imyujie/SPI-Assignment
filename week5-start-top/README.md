##HOW TO BUILD

1. 下载整个文件目录，在根目录下，打开命令行执行如下代码：`npm install`
2. 完成第一步后，执行：`grunt`
3. 完成后，运行服务器：`node server`

##HOW TO RUN

编译过的代码也在目录中，S1~S4 目录下的 `index.js` 为编译后的代码，S1~S4 目录下的 `index.ls` 为编译前的代码。所以可以直接执行：

`node server`

运行并查看效果。
